- ¿Qué es yay?
Yay utiliza un **AUR**, es decir, (Arch User Repository), un repositorio de comandos personalizados de usuarios. 
Yay es un comando para gestionar, descargar e instalar paquetes de este repositorio, **incluido el repositorio de 'pacman'**.
