- ¿Qué es cd?
cd es uno de los comandos más básicos. Permite cambiar el directorio actual en consola.

- ¿Cómo se utiliza y qué opciones tiene?

cd Compendium/COMANDIUM/ <!--Cambia del supuesto directorio home a COMANDIUM-->

cd ~ <!--Cambia al directorio home del usuario actual SIEMPRE-->
cd <!--Sin ninguna opción, hace lo mismo que "cd ~"-->
cd .. <!--Cambia al directorio anterior-->
